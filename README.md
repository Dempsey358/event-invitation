# Event Invitation - インタラクティブWeb招待状

封筒を開けるアニメーションで始まるインタラクティブなイベント招待状。

## Features
- 🎫 封筒オープンアニメーション（紙吹雪付き）
- 📋 イベント詳細表示
- 📅 Googleカレンダー登録ボタン
- ⏳ カウントダウンタイマー
- 📝 出欠回答（Google Apps Script連携）
- 📸 イベント後のフォトギャラリー
- 🎨 4テーマ切り替え

## Setup
1. `config.json` を編集してイベント情報を設定
2. GASセットアップ（`docs/gas-setup.md` 参照）
3. GitHub Pagesにデプロイ
